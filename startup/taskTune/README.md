# Aashu Backend API

Backend API for the Aashu productivity app, built with FastAPI, SQLAlchemy, and OpenAI integration.

## Features

- User authentication with JWT
- Task and subtask management
- AI-powered subtask generation using OpenAI
- RESTful API design
- PostgreSQL database

## Setup

### Prerequisites

- Python 3.8+
- PostgreSQL
- OpenAI API key
